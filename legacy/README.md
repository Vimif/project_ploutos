# Legacy Boundary

This folder contains archived code, configs, and operational helpers that are
kept for historical reference or artifact compatibility.

Nothing here should be treated as part of the supported V9.1 golden path.

Current supported entry points remain:

- `training/train.py`
- `scripts/run_pipeline.py`
- `scripts/robustness_tests.py`
- `scripts/paper_trade.py`
- `scripts/compare_strategies.py`
- `scripts/profitability_audit.py`
- `scripts/run_league_batch.py`

Archived content is kept only when it still helps explain old artifacts,
migration history, or previously used deployment setups.
